# 1. Vision du Produit

## 1.1 Nom du Projet
**Ma Bibliothèque**

## 1.2 Description
Application web personnelle pour gérer sa collection de livres et suivre ses lectures.

## 1.3 Problème
Les lecteurs réguliers oublient quels livres ils ont lus, lesquels ils veulent lire, et rachètent parfois des livres qu'ils possèdent déjà.

## 1.4 Solution
Une application simple pour cataloguer ses livres avec leur statut de lecture.
