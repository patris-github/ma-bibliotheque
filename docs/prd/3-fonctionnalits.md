# 3. Fonctionnalités

## 3.1 Liste des Fonctionnalités (Scope Fixe)

| # | Fonctionnalité | Description |
|---|----------------|-------------|
| F1 | Voir la liste de mes livres | Afficher tous les livres enregistrés avec titre, auteur et statut |
| F2 | Ajouter un livre | Créer une entrée avec titre, auteur et statut de lecture |
| F3 | Modifier un livre | Mettre à jour les informations d'un livre existant |
| F4 | Supprimer un livre | Retirer définitivement un livre de la liste |
| F5 | Filtrer par statut | Afficher uniquement les livres selon leur statut |

## 3.2 Statuts Disponibles

| Statut | Description |
|--------|-------------|
| **À lire** | Livre dans la pile d'attente |
| **En cours** | Livre actuellement en lecture |
| **Lu** | Livre terminé |

## 3.3 Authentification

| Fonctionnalité | Description |
|----------------|-------------|
| Création de compte | L'utilisateur peut créer un compte |
| Connexion | L'utilisateur peut se connecter à son compte |
| Persistance | Les données sont liées au compte et persistent entre les sessions |
