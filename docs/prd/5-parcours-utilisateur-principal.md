# 5. Parcours Utilisateur Principal

## 5.1 Premier Lancement (Nouvel Utilisateur)

```
1. Laure ouvre l'application
2. Elle crée un compte ou se connecte
3. Elle arrive sur une bibliothèque vide
4. Un message d'accueil la guide pour ajouter son premier livre
5. Elle clique sur "Ajouter un livre"
```

## 5.2 Ajouter un Livre

```
1. Laure clique sur le bouton "Ajouter"
2. Elle saisit le titre du livre
3. Elle saisit l'auteur
4. Elle sélectionne le statut (par défaut : "À lire")
5. Elle valide
6. Elle revient à la liste avec le nouveau livre affiché
```

## 5.3 Vérifier en Librairie

```
1. Laure est en librairie, un livre lui plaît
2. Elle ouvre l'application
3. Elle parcourt sa liste ou utilise les filtres
4. Elle vérifie si le livre est déjà dans sa collection
5. Elle décide d'acheter ou non
```

## 5.4 Commencer un Livre

```
1. Laure décide de commencer un nouveau livre
2. Elle ouvre l'application
3. Elle filtre sur "À lire"
4. Elle trouve le livre dans la liste
5. Elle clique pour modifier
6. Elle change le statut en "En cours"
7. Elle valide
```

## 5.5 Terminer un Livre

```
1. Laure termine sa lecture
2. Elle ouvre l'application
3. Elle filtre sur "En cours"
4. Elle trouve le livre
5. Elle clique pour modifier
6. Elle change le statut en "Lu"
7. Elle valide
```
