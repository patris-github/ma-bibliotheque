# 6. Stack Technique

| Composant | Technologie |
|-----------|-------------|
| Frontend | React 18 + Vite 5.x |
| Routing | React Router 7.x |
| UI | Shadcn/ui + TweakCN |
| Backend | Supabase (auth + database) |
| Déploiement | Vercel |

## 6.1 Contraintes Supabase

| Contrainte | Impact |
|------------|--------|
| Pas d'Edge Functions | Logique côté client uniquement |
| Pas de Storage | Pas d'images de couvertures |
| Pas de Realtime | Pas de synchronisation temps réel |
